## 기술 스택

- **React 18** + **Vite** 기반 SPA 구조
- **JavaScript (ES6+)** + **Framer Motion** 애니메이션
- **Three.js** 기반 WebGL 효과 (Galaxy, LightRays, MetallicPaint)
- **Pretendard Variable** 웹폰트 + CSS3 커스텀 스타일링
- **IntersectionObserver API** – 텍스트 등장 애니메이션
- **언어 토글 기능** – `lang-ko`, `lang-en` 클래스 전환 방식
- **React Router v6** – 페이지 기반 언어 구분
- **반응형 웹** – PC/Mobile 대응
- **AWS EC2 (Ubuntu 22.04)** + **Nginx** 서버 구성
- **Cloudflare DNS / SSL**

## 디자인 의도

* "읽히는 디자인"을 목표로 폰트, 행간, 색상 대비 설정
* 일반적인 템플릿이 아닌 **CLI 느낌의 문서형 인터페이스**
* 프로필 팝업, 텍스트 강조 효과, 깃허브 링크 등의 요소는 **최소한의 인터랙션으로 구성**
* 라이트한 애니메이션과 시각적 밀도를 조절해 **몰입감을 주되 방해는 하지 않음**

## 로고 해석

![L 로고](https://github.com/user-attachments/assets/227cc3d5-64cd-4021-9098-c74a073c506c)

이니셜 **L / H / J**를 수직·수평·사선으로 표현한 기하학적 조형.

* L/H: 축적된 지식과 경험의 시간성
* J: 도전의 순간, 진로 변경의 의지
* 교차점: 기술과 관점이 만나는 전환점
* 컬러: #03ff95 포인트 (인터랙션 시에만 등장)
